# vibe-check

Run a vibe check on your project.

## Install

```
/plugin install github:buckle42/vibe-check
```

## Usage

```
/vibe-check
```

## What You Get

- Code stats (lines, files, breakdown by type)
- Git stats (commits, contributors, streak)
- Activity calendar (GitHub-style heatmap)
- Commit hours heatmap
- Trophy case
- Vibecoding vs Human comparison
- Nerd culture fun facts (Star Wars, LOTR, Terminator, etc.)
- Random encouragement
